# orpheusdark.github.io

Personal portfolio website for Nirant Chavda — Computer Engineering Student focused on cybersecurity, fraud detection, and real-world problem solving.

## Tech Stack

- **Next.js 16** (Static Export for GitHub Pages)
- **Tailwind CSS v4**
- **Framer Motion**
- **TypeScript**

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
# Static output in /out directory
```

## Sections

1. **Hero** — Animated full-screen intro
2. **About** — Story and background
3. **Projects** — RIFT-26 (featured) + 8 other projects
4. **Skills** — Animated skill tags and tech stack
5. **Experience** — Hackathon achievements
6. **Certifications** — Collapsible certification list
7. **Contact** — Social links + contact form
